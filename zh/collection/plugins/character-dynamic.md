# Plugin-Character-Dynamic

## 概述

一个动态特征文件加载插件，根据特定的链上信息自动更新 Eliza 的角色文件。这使得 AI 代理能够根据区块链事件和状态调整其行为和特征。

## 功能特性

- **动态角色更新**：实时角色文件修改
- **链上事件监控**：基于区块链状态的适应
- **自动行为调整**：自我进化的角色特征
- **状态驱动的个性**：对区块链响应的个性变化
