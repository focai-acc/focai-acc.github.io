# 快速入门指南

## 环境要求

开始之前，请确保已安装以下软件：
- Python 2.7+
- Node.js 23+
- pnpm
- Git
- 代码编辑器（推荐使用 VS Code）

## 开发环境设置

### 1. 克隆仓库

```bash
git clone https://github.com/focai-acc/focEliza.git
cd focEliza
```

### 2. 安装依赖

```bash
pnpm install
```

### 3. 配置环境

```bash
cp .env.example .env
# 编辑 .env 文件设置你的配置
```

### 4. 启动开发服务器

```bash
pnpm dev
```
