## Getting Started

### Prerequisites
- Python 2.7+
- Node.js 23+
- pnpm

### Quick Start
```bash
git clone https://github.com/elizaos/eliza-starter.git
cd eliza-starter
cp .env.example .env
pnpm i && pnpm build && pnpm start
```
